# Milestone 00: Keyboard Foundation

## What changed
- The generic MathLive keyboard has been replaced with curated Calcwiz pages.
- Only `Core`, `Letters`, and `Greek` are visible.
- Duplicate-looking symbols were reduced by moving alternates to long-press variants.

## What the pages mean
- `Core`: digits, operators, fraction, square root, powers, `Ans`, `π`, `e`
- `Letters`: everyday symbolic variables such as `x`, `y`, `a`, `b`, `c`, `n`
- `Greek`: symbolic Greek variables only

## Where to find important symbols
- Fraction: `Core`
- Square root: `Core`
- Power: `Core`
- `Ans`, `π`, `e`: `Core`
- `x`, `y`, `a`, `b`, `c`, `n`: `Letters`
- `α`, `β`, `γ`, `θ`, `λ`, `μ`, `Σ`: `Greek`

## Variants
- Long-press `ε` for `ϵ`
- Long-press `θ` for `ϑ`
- Long-press `ρ` for `ϱ`
- Long-press `φ` for `ϕ`
- Long-press `Σ` for `σ` and `ς`

## Worked examples
1. Build `1/3 + 1/6`
   - Open `Core`
   - Tap `a/b` twice
   - Fill the placeholders
   - Press `EXE`
2. Insert a Greek variable
   - Open `Greek`
   - Tap `α`
   - Return to `Core` for operators
3. Reuse a result
   - Solve an expression
   - Use `Copy Result` or `To Editor`
   - Continue editing from there

## Common mistakes
- `Σ` here is only a letter in this milestone, not the discrete summation operator.
- The virtual keyboard inserts math notation; solving and transformations still use the app soft keys and keypad.
- Use `Letters` and `Greek` for variable entry instead of expecting every alphabet symbol at once.

## Exact vs numeric
- Textbook math entry stays structured.
- Exact and numeric output still depends on the active app mode and soft-key action.

## Best modes for this milestone
- `Calculate` for expressions
- `Equation` for solving equations in `x`
- `Table` for function entry with the simpler `Core` + `Letters` subset
